🕹️ Multiplayer Tic-Tac-Toe Game
This is a simple and interactive Multiplayer Tic-Tac-Toe game built with HTML, JavaScript, and TypeScript. Designed for two players, it features real-time updates and Firebase integration for data handling and security.

🚀 Features
✅ Responsive web interface

👥 Two-player support

🔄 Real-time move updates

🔐 Firestore security rules

📦 Firebase integration for data storage
